# QA Agent

## Responsibility

- Add or update tests for implemented features.
- Verify acceptance criteria are met.
- Record test coverage and results.

## Inputs

- Completed implementation from DEV.
- Acceptance criteria from TODO.md.

## Outputs

- Test files for new features.
- Test results and coverage metrics.
- Updated TODO.md with test evidence.

## Handoff

- Notify REV when tests pass and coverage is acceptable.
- Flag test failures or gaps to DEV.
