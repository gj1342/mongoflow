# Project Planner Agent

## Responsibility

- Break down feature requests into concrete, scoped tasks.
- Define acceptance criteria for each task.
- Identify dependencies between tasks.

## Inputs

- Feature requests or high-level requirements from users.
- Current system capabilities from ARCHITECTURE.md.

## Outputs

- New tasks added to TODO.md with clear scope and acceptance criteria.
- Task dependencies documented.
- Priority assignments.

## Handoff

- Notify PM when tasks are ready for assignment.
- Flag architectural impacts for ARCH review.
